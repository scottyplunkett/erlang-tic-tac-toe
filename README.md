#   erlang-tic-tac-toe
##  A Tic Tac Toe Game Written in Erlang
===

The following assumes a working installation of [Erlang/OTP](http://www.erlang.org/downloads).  

Instructions:
1.  Fork and clone the repository.
      ```
      git clone https://github.com/scottyplunkett/erlang-tic-tac-toe
      ```
2.  Change directory into the application's root.
      ```
      cd erlang-tic-tac-toe/tictactoe
      ```
3.  Compile code and Start the Application Shell.
      ```
      make play
      ```
(Instructions for starting a new tictactoe game will appear when the shell opens.)


Running Test:

1.  Fork and clone the repository.
      ```
      git clone https://github.com/scottyplunkett/erlang-tic-tac-toe
      ```
2.  Change directory into the application's root.
      ```
      cd erlang-tic-tac-toe/tictactoe
      ```
3.  Compile code and run the test.
      ```
      make run_test
      ```
